# bobobears's Dotfiles

Ubuntu 系统配置备份，用于快速恢复开发环境。

## 包含内容

| 目录 | 说明 |
|------|------|
| shell/ | .bashrc, .profile, .gitconfig |
| hermes/ | Hermes Agent 配置(config.yaml, SOUL.md) |
| ssh/ | SSH 客户端配置(config) + 公钥 |
| packages/ | 系统包列表 |
| gnome/ | GNOME 桌面/快捷键设置 |
| apps/ | LM Studio, AppImage 快捷方式等 |
