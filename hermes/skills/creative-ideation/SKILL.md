# Creative Ideation — Constraint-Driven Project Generation

Generate creative project ideas by applying structured constraints to break out of typical patterns.

## Method

Apply constraints from the following dimensions (mix 2-3 per ideation session):

### 1. Medium Constraints
- CLI only, browser only, mobile-first
- No external dependencies
- Runs offline completely
- Must fit in one file

### 2. Aesthetic Constraints
- Retro (8-bit, terminal aesthetics)
- Minimal (single color, no images)
- Generative (different every time)
- Text-only, ASCII art

### 3. Function Constraints
- Single purpose
- Does one thing beautifully
- Solves a personal annoyance
- Automates a tedious manual task

### 4. Time Constraints
- Build in 30 minutes
- Build in one sitting
- Weekend project

## Process

1. Gather the user's interests and pain points
2. Pick 2-3 constraint dimensions
3. Generate 3-5 ideas combining constraints + interests
4. Present with the "why" for each idea
5. Let the user pick one to explore further
