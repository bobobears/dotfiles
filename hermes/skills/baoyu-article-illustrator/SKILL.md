# Article Illustrator

Generate consistent illustrations for articles. Maintains visual coherence across an article's illustrations by controlling type, style, and palette.

## When to Use

- User wants illustrations for their article or blog post
- User needs featured images or inline illustrations
- User wants consistent visual style across multiple images

## Features

- **Type control**: diagram, photo-realistic, cartoon, watercolor, sketch, infographic
- **Style consistency**: same style across multiple illustrations in one series
- **Palette control**: specify color palette for visual coherence
- **Composition**: portrait, landscape, square, hero image

## Workflow

1. Read the article understanding the content and key visual moments
2. Discuss with user the illustration type and style preferences
3. Generate illustrations section by section with consistent parameters
4. Review and adjust as needed

## Quality Guidelines

- Illustrations should match the article's tone
- Maintain character consistency across multiple illustrations in the same article
- Use appropriate level of detail (not too busy for small images)
- Consider text placement (avoid covering important elements)
