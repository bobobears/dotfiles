# Pixel Art Generator

Generate pixel art with authentic retro console palettes.

## Palettes

| Era | Colors | Use Case |
|-----|--------|----------|
| NES | 64 colors | Classic Nintendo style |
| Game Boy | 4 shades | Monochrome retro |
| PICO-8 | 16 colors | Fantasy console aesthetic |
| SNES | 256 colors | 16-bit era style |

## Usage

Generate pixel art by specifying:
- Subject (e.g., "dragon", "forest scene", "space ship")
- Palette (NES / Game Boy / PICO-8 / SNES)
- Size (e.g., 16x16, 32x32, 64x64)
- Style (sprite, tile, character, scenery)

Design a prompt that describes:
1. The subject and composition
2. The palette to use
3. The pixel dimensions
4. Any specific retro style reference

Render as HTML canvas, SVG, or image file.
